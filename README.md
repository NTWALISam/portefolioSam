# NTWALI MUTAONA Samuel — Portfolio Website (Static)

A ready-to-deploy, single-page portfolio built from your CV.

## Quick start

### Option A — Open locally
1. Download the project
2. Open `index.html` in your browser

### Option B — GitHub Pages
1. Create a GitHub repository (e.g. `ntwali-portfolio`)
2. Upload all files (keep the folder structure)
3. In GitHub: **Settings → Pages → Build and deployment → Deploy from a branch**
4. Select branch `main` and folder `/root`

### Option C — Netlify (drag & drop)
1. Go to Netlify → Add new site → Deploy manually
2. Drag & drop the project folder

## What to edit
- `index.html`
  - Replace the `LinkedIn` link (`href="#"`) with your real URL.
  - If you want, add a profile photo in `assets/img/` and reference it in the hero.

## Contact form (optional)
This site ships with a static form.
- If you deploy on Netlify, you can enable **Netlify Forms**.
- Or you can use Formspree and replace the form action.

## Files
- `index.html` — main page
- `assets/css/styles.css` — styling
- `assets/js/main.js` — mobile menu
- `assets/docs/NTWALI_MUTAONA_Samuel_CV.pdf` — your CV
